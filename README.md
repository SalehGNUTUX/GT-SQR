## 🌟 مميزات المشروع

### 📖 التلاوة والمحتوى
- اختيار أي سورة وأي نطاق من الآيات
- دعم 6 قراء مشهورين (العفاسي، السديس، المنشاوي، الحصري، الشاطري، الماهر)
- تحميل الصوت تلقائياً من EveryAyah.com
- دعم الترجمة: ميسر، جلالين، Sahih English، Pickthall، Français، اردو
- مقاسات ريلز: 9:16 (تيك توك/ريلز) • 16:9 (يوتيوب) • 1:1 (انستجرام)

### 🎬 المشهد والخلفية
- 8 ثيمات جاهزة (زمرد، ذهبي، ليلي، وردي، محيط، صحراء، بنفسجي، أسود)
- خلفية تدرج بألوان قابلة للتخصيص واتجاهات متعددة
- رفع صورة أو فيديو كخلفية
- حركات الخلفية: ثابت / زحف سينمائي / تكبير / تحريك أفقي
- 4 فلاتر لونية: طبيعي / أبيض وأسود / دافئ / بارد

### 🕌 الزخارف الإسلامية
- شبكة سداسية • هندسة إسلامية • نجوم ثمانية • قوس محراب • إطار منقوش

### ✍️ النصوص والخطوط
- 9 خطوط عربية جاهزة: Amiri Quran • Reem Kufi • Scheherazade • Cairo • Noto Naskh • Lateef • Harmattan • Markazi Text • Aref Ruqaa
- **رفع خطوط مخصصة** (TTF / OTF / WOFF / WOFF2)
- 6 تأثيرات نصية: بدون / توهج ذهبي / نيون / حدود / ظل 3D / منقوش
- 4 تأثيرات ظهور الآيات: مباشر / تلاشي / انزلاق / تكبير
- علامة مائية مخصصة بمواضع متعددة

### 🎵 المكساج الصوتي
- التحكم بمستوى صوت القارئ
- رفع موسيقى/أناشيد خلفية (MP3/AAC/OGG)
- موجات صوتية متحركة: أعمدة / موجة / نقاط
- تحكم كامل في مستوى الصوت وموضع الموجة

### ⚡ التأثيرات المتقدمة (جديد)
- نتوء (Vignette) • حبوب فيلم • إطار ذهبي متوهج
- نجوم متلألئة • أشعة نورانية • ضبابية بوكيه
- 4 أنواع انتقالات بين الآيات

### 📂 القوالب
- حفظ إعدادات كاملة كقالب مسمى
- تطبيق القوالب بنقرة واحدة
- تخزين القوالب محلياً في المتصفح

### 📥 التصدير
- تصدير WebM (كروم/فايرفوكس)
- تصدير MP4 (إذا دعمه المتصفح)
- شريط تقدم لحظي أثناء التصدير

### 🌐 يعمل أونلاين وأوفلاين
- **PWA**: قابل للتثبيت كتطبيق
- **Service Worker**: تخزين مؤقت للملفات الأساسية
- **وضع غير متصل**: يتعرف على حالة الاتصال تلقائياً

## 🚀 التشغيل

### تشغيل محلي (بدون خادم)
```bash
git clone https://github.com/SalehGNUTUX/GT-SQR.git
cd GT-SQR
# افتح index.html مباشرة في المتصفح
# أو استخدم خادم محلي بسيط:
python3 -m http.server 8080
# ثم افتح: http://localhost:8080
```

### تشغيل عبر الإنترنت
افتح الرابط المباشر أو انشره على:
- GitHub Pages
- Netlify
- Vercel
- أي استضافة ثابتة

## 🔌 المصادر المستخدمة

| المصدر | الاستخدام | الرابط |
|--------|-----------|--------|
| AlQuran Cloud API | بيانات الآيات والترجمات | [api.alquran.cloud](https://api.alquran.cloud) |
| EveryAyah.com | تلاوة القراء الصوتية | [everyayah.com](https://everyayah.com) |
| Google Fonts | الخطوط العربية | [fonts.google.com](https://fonts.google.com) |

| Canvas API | رسم الإطارات | متصفح مدمج |
| Service Worker | دعم العمل أوفلاين | متصفح مدمج |
